# 图片资源目录

此目录用于存放小程序所需的图片资源。

## 需要的图片

### TabBar图标（需要创建）

请创建以下图标文件，建议尺寸：81x81px：

- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `base.png` - 基地图标（未选中）
- `base-active.png` - 基地图标（选中）
- `qrcode.png` - 二维码图标（未选中）
- `qrcode-active.png` - 二维码图标（选中）
- `profile.png` - 我的图标（未选中）
- `profile-active.png` - 我的图标（选中）

### 图标设计建议

- 使用绿色主题色（#4ade80）
- 未选中状态：灰色（#666666）
- 选中状态：绿色（#4ade80）
- 格式：PNG，透明背景

### 临时方案

如果暂时没有图标，可以：
1. 使用在线图标生成工具
2. 或者暂时注释掉 `app.json` 中的 `tabBar` 配置
3. 使用文字导航代替图标
