# 采摘工端小程序

与 app 前端采摘工端功能一致，已对接后端 API。

## 功能

- **广场**：推荐基地、用户统计（已做天数、待收工资）、查看岗位、立即报名
- **我的报名**：报名进度、待发放核对、确认工资
- **签到码**：考勤签到二维码（后端 API 生成）
- **我的**：基本信息、薪资卡号、工作历程、设置

## 编译运行

1. 用**微信开发者工具**打开 `miniprogram` 目录
2. 确保后端已启动：`cd app/backend && npm run start:dev`
3. 在开发者工具中：**详情 → 本地设置 → 勾选「不校验合法域名、web-view...」**
4. 编译并预览

## API 配置

`app.js` 中 `baseUrl` 默认 `http://localhost:3001/api`。若后端地址不同，请修改。

## 二维码说明

签到码使用 `api.qrserver.com` 生成二维码图片。若无法加载：
- 开发时确保已勾选「不校验合法域名」
- 正式发布需在微信公众平台将 `api.qrserver.com` 加入「request合法域名」
